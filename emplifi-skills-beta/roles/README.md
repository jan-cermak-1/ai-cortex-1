# Emplifi Roles

> Role definitions for AI assistants — Product Manager, Product Designer, Developer

## Roles

| File | Description |
|------|-------------|
| [product-manager.md](product-manager.md) | PM — feature design, specs, integration |
| [product-designer.md](product-designer.md) | Product Designer — UI, SOUL, design patterns |
| [developer.md](developer.md) | Developer — implementation, SOUL components, API |

## Usage

Add the relevant role to Cursor rules or reference it in chat:

- `@emplifi-skills-beta/roles/product-manager.md`
- `@emplifi-skills-beta/roles/product-designer.md`
- `@emplifi-skills-beta/roles/developer.md`

Each role references the appropriate context (context/, soul-design-system/).
