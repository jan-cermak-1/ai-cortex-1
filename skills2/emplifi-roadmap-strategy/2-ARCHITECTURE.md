# Emplifi Platform Architecture

> 7-layer platform model and product groups taxonomy

## 7-Layer Platform Model (Client Deck)

From "Client event recording / Product Platform Overview":

### Layer 1: Governance
- Authentication
- Team/Org Setup
- Access Control
- Compliance
- Data Encryption
- Threat Detection

### Layer 2: Analytics
- Templates
- Self-service Boards
- Anomaly Detection
- Collaboration
- Notifications
- Data Sharing

### Layer 3: Platform Intelligence & Automation
- Scheduling
- Workflow Automation
- Campaigns
- Influencers
- Data Outreach

*In practice: PIE + Fuel + orchestration across the suite*

### Layer 4: Content
- Content Management
- Consent Management
- Collaboration
- Adjustments
- Recommendations
- Storage

### Layer 5: AI & Machine Learning
- Image & Video analysis
- Generative AI
- Text/NLP
- Recommendations
- Machine Learning

### Layer 6: Content Distribution
- Publishing
- Surveys
- Live Streaming
- Website
- Calls
- Emails
- Partners
- Client-specific systems

### Layer 7: Bi-Directional Data Integration
- Public Social
- Private Social
- Reviews
- Messaging
- Calls
- Emails
- Webchat
- Client Specific APIs
- Direct Input

### Cross-cutting Entities
- **Customer**
- **Product**

This describes Emplifi as a full platform with governance, analytics, AI, content, distribution and integrations - not just a few UI modules.

## 2023 Roadmap Product Groups Taxonomy

From Emplifi 2023 Roadmap:

### Care Group
- Digital Care
- Community Management
- Conversational / Chatbot
- Live web chat
- Reviews moderation
- Voice of Customer
- Agent
- CRS
- Agent Analytics
- EVA (Emplifi Virtual Assistant - legacy name for bot/assistance features)
- Knowledge Base

### Data Insights Group
- Organic Social Analytics
- Paid Social Analytics
- Platform Dashboard
- Web Analytics
- Unified Analytics
- Digital Listening

### Content Creation Group
- Content Publishing
- Link in Bio
- Influencers / Creators
- Content Inspiration (e.g. AI Composer)
- Asset Management

### Commerce Group
- Social UGC
- Ratings & Reviews
- 1:1 Live Advisor
- 1:N Live Streaming
- Experience Layer (Unified CTA on web)

### Service Center Group
- Service workflows
- Contact center integrations (e.g. Five9)
- CRM routing

### Shared Platform Group
- Platform Intelligence Engine (PIE)
- Public Platform APIs
- Productized Connectors
- Product/Account User Permissions
- Security & Token Flow
- Notifications
- Multi-cloud & performance
- Native mobile apps
- Butler / support interface
- Payment Gateway
- Platform Settings Module
- Internal tools integrations
- Design System
